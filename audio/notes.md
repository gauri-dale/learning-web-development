
# To add Audio files

- Use a `<audio> </audio>` tag.
- Between the audio tags, there is a `<source>` element. \

`<source src = " ">`

# Attributes 
- The song won't play because of lack of `controls` \
`<audio controls>` \

- `autoplay` plays song automatically. \

- `muted` the song is muted unless the user unmutes it.

- `loop` loops the song
